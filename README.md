# Graphical-EDA-using-Python

## Overview

This project provides a comprehensive visual exploration of historical motor sales data. By analysing trends across different vendors, locations, and product types, we uncover insights to inform strategic business decisions.

## Objectives

Using historical motor sales data, we aim to identify key sales patterns and predict future trends. By leveraging different visualisation techniques, we provide actionable insights for boosting sales performance.

## Data and Tools

The dataset is an Excel file containing vendor code, plant location, type, inventory number, and other sales metrics. The analysis was conducted using Python with libraries including Pandas, NumPy, Matplotlib, Seaborn, and Plotly. The data was initially collected as a CSV and then transformed into Excel for processing.

## Process

- **Data Collection:** Gathered historical sales data in a CSV file.
- **Data Processing:** Cleaned and transformed the data into a structured Excel sheet.
- **Exploratory Data Analysis:** Conducted descriptive statistics, distribution plots, and summary tables.
- **Visualisation:** Created bar charts, heatmaps for comparative analysis, and line plots for time series trends.
- **Business Optimisation:** Derived insights and recommendations for boosting future sales.

## Conclusion

The motor sales analysis offers a rich visual perspective on historical data, identifying patterns and anomalies that can guide strategic business decisions and future growth.
